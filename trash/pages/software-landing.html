My Projects

Pong Game in Java

My own version of the game of Pong implemented in Java through the Processing platform. The game includes a normal mode, a gravity mode, and a random pattern mode (shown in image). Check it out on GitHub!

Brandeis Course Catalog

A site built on Ruby on Rails where the user can search a database containing courses offered at Brandeis University. The purpose was to learn how to implement authentication, user login, using an MVC architecture. I also learned AJAX for this excersise. You can check it out here!

Personal Website

Personal website that I built using a library from HTML5 UP. The site has software development projects, photographic projects, some of my Physics research, and other interesting things that I've done. This is a work in progress. The source code can be seen here.

Cloud IDE for Programming Assignments (University Courses)

A web application to seamlessly connect professors, teaching assistants, and students, allowing for an accelerated learning in computer science classes. AspirinX is programming assignments in the cloud. This is a web-based application being developed by a team of four people using Ruby on Rails and some JavaScript. You can check out the ongoing project on our GitHub repository! The project will be finished by the end of December 2017.

The Monty Hall Show - Quick Coding Challenge

A one-day coding project that I did as a way of practicing HTML, CSS, JavaScript and JQuery after I taught them to myself. Check out the project online or the see the code at my Github repository.

Simulation of a Flexible Polymer's Random Walk

A Physics research project with strong computational elements, where we tried to simulate the behavior of flexible polymers, in an attempt to extract relevant statistics that might help us predict future polymer behavior. The simulation itself is a Java program. The user can specify in the code the number of monomers (units) that the polymer will have, and the number of simulations that the program will be run. Then important statistics are calculated and written in a file specified by the user. See the project's GitHub here.

React-Ion
